## Installing Git

Git is a version control system used to track changes in files and manage different versions of a project. It helps developers save their work, collaborate with others, and maintain a history of project changes.

To install Git on Ubuntu, open the terminal and run the following command:

```bash
sudo apt install git
```

**Explanation:** This command installs Git from Ubuntu's software repositories using administrator privileges.





## GitHub

GitHub is a cloud-based platform used to store, manage, and share Git repositories. It allows developers to back up their projects, track changes, and collaborate with others.

To verify that Git has been installed successfully, open the terminal and run the following command:

```bash id="k7v2m4"
git --version
```

**Explanation:** This command displays the installed Git version on the system.
