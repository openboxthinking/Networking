#INSTALLATION OF GIT IN Ubuntu

Step 1 — Update your system first
bash      
sudo apt update

Step 2 — Install Git
bash
sudo apt install git


Step 3 — Verify installation
bash
git --version

You should see something like:
git version 2.34.1

Step 4 — Configure Git (Important — do this right after install)
bash
git config --global user.name "Your Name"
git config --global user.email "youremail@gmail.com"

Replace with your actual name and email. This is used to track your commits.

Step 5 — Confirm your config
bash
git config --list

That's it. Git is installed and ready.
